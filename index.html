1 Project Summary

This project uses ARIMA modeling to forecast Mexico’s Consumer Price Index (INPC) from 2010 to 2015. It includes data transformation, model selection, diagnostics, and 5-month forecasting, all done in R.

9 Residual Diagnostics

res <- modelo3$residuals
par(mfrow = c(1,1))
plot(res, main = "Residuals of ARIMA(2,1,2)")

qqnorm(scale(res)); qqline(scale(res))

ks.test(scale(res), "pnorm")

10 Forecasting

final_model <- Arima(INPC.ts, order = c(2,1,2))
forecast_result <- forecast(final_model, h = 5)
plot(forecast_result, xlab = "Date", ylab = "Forecasted INPC", main = "5-Month Forecast")
lines(final_model$fitted, col = "blue", lty = 2)

11 Conclusion

The ARIMA(2,1,2) model fits the INPC data well, with white noise residuals and statistically significant coefficients. Forecasts suggest a slight upward trend in the price index, consistent with macroeconomic expectations.
